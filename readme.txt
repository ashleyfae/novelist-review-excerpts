== Description ==

Allows you to include excerpts of reviews on your book pages.

== Installation ==

1. Upload `novelist-review-excerpts` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add review excerpts to your book layout in Books -> Book Layout.
1. Add or edit a book and start adding review quotes.
